# **Data Analysis of Titanic Dataset**
This project explores the Titanic dataset using Python to investigate the factors that influenced passenger survival. The analysis aims to examine whether a passenger's likelihood of survival was affected by their socioeconomic status, embarkation port, demographic characteristics, fare category, and age group.

Through exploratory data analysis and visualization, the project seeks to identify patterns and relationships between survival outcomes and variables such as:

* Passenger class (Pclass) – representing socioeconomic status.
* Embarkation town – where passengers boarded the ship.
* Demographic groups – based on sex and family composition.
* Fare category – grouped according to ticket prices.
* Age category – classified into different age groups.

The objective is to gain insights into the social and economic factors that influenced survival during the Titanic disaster.
